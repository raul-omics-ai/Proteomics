Instalación y uso de MaxQuant en Windows

Este documento describe cómo preparar un sistema Windows para ejecutar
MaxQuant correctamente y cómo iniciarlo.

------------------------------------------------------------------------

Requisitos del sistema

-   Windows 10 u 11 (64 bits) o Windows Server 2016/2019/2022
-   .NET 8.0 o superior (recomendado: 8.0.302 o más reciente)
-   MaxQuant descargado y descomprimido en una carpeta local

  Nota: .NET 3.1 y versiones anteriores no están soportadas.

------------------------------------------------------------------------

1. Comprobar si .NET está instalado

1.  Pulsa Win + R
2.  Escribe cmd y pulsa Enter
3.  En la ventana de comandos escribe:

    dotnet --version

-   Si aparece algo como 8.0.xxx, ya tienes .NET instalado.
-   Si aparece un error del tipo “‘dotnet’ no se reconoce como un
    comando”, debes instalarlo.

También puedes comprobar si existe la carpeta:

    C:\Program Files\dotnet\sdk

Si no existe o está vacía, necesitas instalar .NET.

------------------------------------------------------------------------

2. Instalar .NET 8

1.  Ve a la página oficial de Microsoft:
    https://dotnet.microsoft.com/en-us/download/dotnet/8.0
2.  En la sección “.NET SDK 8.0”, descarga:
    -   Windows x64 Installer
3.  Ejecuta el instalador y sigue los pasos habituales.

Cuando termine la instalación:

-   Abre de nuevo cmd
-   Ejecuta:

    dotnet --version

Deberías ver algo como 8.0.3xx.

------------------------------------------------------------------------

3. Ejecutar MaxQuant

Según el README oficial de MaxQuant:

Manual / Demo

-   Para ejecutar MaxQuant, haz doble clic en StartMaxQuant.bat dentro
    de la carpeta de MaxQuant y especifica los parámetros de tu
    experimento en la interfaz gráfica (GUI).
-   Para tutoriales más detallados:
    -   Artículo de Nature Protocols con una descripción paso a paso
        para distintos diseños experimentales:
        https://www.nature.com/articles/nprot.2016.136
    -   Canal de YouTube de MaxQuant con tutoriales en profundidad:
        https://www.youtube.com/c/MaxQuantChannel

Resultados

-   MaxQuant genera los archivos de salida en la carpeta:

    combined/txt

-   Esta carpeta contiene distintos informes, incluyendo cuantificación
    a nivel de proteínas, péptidos y PTMs.

Tiempo de ejecución

-   El tiempo de ejecución puede variar ampliamente dependiendo de:
    -   La configuración experimental
    -   El número de archivos
    -   El hardware disponible

Como referencia: - El análisis de una muestra LFQ típica (1 archivo
RAW), digerida con tripsina y usando un FASTA humano, en un ordenador de
sobremesa “normal”, suele tardar aproximadamente 1 hora.
